# PCA Dimensionality Reduction Example

This example demonstrates Principal Component Analysis (PCA) by reducing synthetic 3-dimensional data to 2 dimensions. It shows how PCA transforms high-dimensional data into a lower-dimensional representation while preserving most of the original variance. The output includes the original and transformed data shapes, and the explained variance ratio.

## Language

`python`

## How to Run

1. Ensure you have Python, numpy, and scikit-learn installed (`pip install numpy scikit-learn`).
2. Save the code as `main.py`.
3. Run from your terminal: `python main.py`
